üèõÔ∏è About Ol Chiki Script

Pandit Raghunath Murmu (1905‚Äì1982) created the Ol Chiki script in 1925 to give the Santali language its own writing system. Before Ol Chiki, Santali was written in various borrowed scripts (Bengali, Devanagari, Oriya, Roman).

The name "Ol Chiki" means "writing" in Santali. The script was designed to represent all the unique sounds of the Santali language that other scripts couldn't capture properly.

Ol Chiki is now used officially in Jharkhand, Odisha, and West Bengal. It was added to Unicode in version 5.1 (2008).
